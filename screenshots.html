Screenshots of User interfaces

Admin dashboard

Below list showcase some of the features of the admin dashboard:

  • Add new flight
  • Update existing flight
  • Manage passenger records & more

Passenger dashboard

Below list showcase some of the features of the passenger dashboard:

  • View list of available flights
  • Book a new flight ticket
  • Check flight status, cancel ticket & more

Welcome page

Below list showcase some of the features of the welcome page:

  • Displays Book now and Search flights buttons
  • About Us section, Contact Us, FAQ and other stuff are displayed on this page

Search flights feature

Below list showcase some of the features of the search flights feature:

  • Accepts source, destination and date of journey from the passenger
  • Redirects the passenger to the search results page

Search results page

Below list showcase some of the features of the search results page:

  • Shows the list of available flights as per search inputs
  • If no flights are available, displays an error message

Passenger login page

Below list showcase some of the features of the passenger login page:

  • Takes in passenger ID and password from the passenger
  • Verifies the details with the database

Passenger signup page

Below list showcase some of the features of the passenger signup page:

  • Accepts details like passenger ID, password, name, contact no. etc. from the passenger
  • Checks if the passenger ID or email ID already exists in database or not
  • Data type checks are used at appropriate places

Admin login page

Below list showcase some of the features of the admin login page:

  • Takes in admin ID and password from the admin
  • Verifies the details with the database

Passenger profile page

Below list showcase some of the features of the passenger profile page:

  • Gives choice to the passenger to update his/her details

Book ticket page

Below list showcase some of the features of the book ticket page:

  • Takes source, destination etc. from the passenger
  • Shows the list of available flights
  • Passenger can then proceed to the payment form

Make payment page

Below list showcase some of the features of the make payment page:

  • Accepts payment details from the passenger
  • Sets the reservation status of passenger as waiting so that the admin can confirm it and update it

View previous bookings page

Below list showcase some of the features of the view previous bookings page:

  • Shows the list of all previous bookings made by the passenger

Flight Status page

Below list showcase some of the features of the flight status page:

  • Shows the flight status of the booked flight to the passenger

Partner Airlines page

Below list showcase some of the features of the partner airlines page:

  • Shows the logo of all the airlines that are currently providing services

Update flights feature

Below list showcase some of the features of the update flights feature:

  • It allows the admin to update details of an existing flight

Remove flight feature

Below list showcase some of the features of the remove flight feature:

  • Allows the admin to delete an existing flight permanently from the system

Manage booked flight tickets

Below list showcase some of the features of the manage booked flights feature:

  • Allows the admin to update reservation status of booked flight tickets

Manage Passenger Records

Below list showcase some of the features of the manage passenger records feature:

  • Shows the details of all passengers registered into the system
  • Allows admin to delete a passenger record permanently
